# ⏱️ Realistic Timelines (3 / 6 / 12 / 18 / 24 Months)

> All timelines assume **~20 hrs/week** and that you're a **Junior Software Engineer** (you already code, use Git, debug, read docs). Adjust ±30% for prior math/ML exposure.

[⬅ Back to README](../README.md)

---

## How to Read These

- **"Job-ready"** = you can pass interviews and contribute on day one for that role, with a portfolio to prove it.
- Each timeline is **cumulative** — the 6-month plan includes the 3-month plan, etc.
- The fast track **front-loads applied GenAI** (Phases 5–8) because that's where 2026 hiring demand is highest, then backfills fundamentals. See [80/20 Skills](17-skills-mistakes-interview.md).

---

## 🟢 3 Months — "Dangerous with GenAI" (Applied Fast Track)

**Goal:** Be able to build and ship real LLM apps (RAG + basic agents). Land junior/contract Applied AI work or internal projects.

| Weeks | Focus | Phases | Deliverables |
|-------|-------|--------|--------------|
| 1 | Setup + landscape | 0 | LLM Hello World CLI, journey repo |
| 2–4 | Python for AI + data stack (refresh) | 1 | DataOps toolkit, async LLM batch processor |
| 5–6 | GenAI essentials + embeddings + HF | 5 (applied) | Semantic search, multi-provider chat |
| 7 | Prompt engineering + structured output + eval | 6 | Prompt eval lab |
| 8–10 | **RAG** (build 2–3 RAG projects) | 7 | "Chat with PDFs", docs assistant, eval harness |
| 11–12 | **Agents** (single-agent + tools + MCP) | 8 (intro) | ReAct assistant, an MCP toolbox |

**Outcome:** 4–6 shipped projects, comfortable with OpenAI/Anthropic/Ollama, RAG, and basic agents. *Math/DL deferred.*

## 🔵 6 Months — **Job-Ready as an AI / GenAI Engineer** ⭐ (Recommended Fast Track)

**Goal:** Pass Applied AI / GenAI Engineer interviews with a strong portfolio.

| Month | Focus | Phases | Key Deliverables |
|-------|-------|--------|------------------|
| 1 | Setup + Python/DSA/SQL | 0, 1 | Toolkit, SQL analytics |
| 2 | Math intuition (parallel) + ML | 2, 3 | Regression from scratch, end-to-end ML service |
| 3 | Deep Learning + Transformers | 4 | PyTorch fine-tuner, self-attention from scratch |
| 4 | GenAI + Prompt Engineering | 5, 6 | GenAI playground, prompt eval lab |
| 5 | **RAG (all 5 projects)** | 7 | Hybrid+reranked+evaluated RAG |
| 6 | **Agents + Deployment basics** | 8, 11 (intro) | 3–4 agent projects, 1 productionized app |

**Outcome:** Job-ready for **AI Engineer / GenAI Engineer / Applied AI Engineer**. 10–12 portfolio projects, 2–3 polished flagships. Start applying in month 5.

## 🟣 12 Months — Well-Rounded AI Engineer (Specialist-Ready)

**Goal:** Strong across the stack, beginning to specialize (LLM Eng or MLOps).

| Months | Focus | Phases |
|--------|-------|--------|
| 1–3 | Foundations + ML + DL | 0–4 |
| 4–6 | GenAI + Prompt + RAG + Agents | 5–8 |
| 7–8 | **Fine-tuning** (LoRA/QLoRA/DPO) | 9 |
| 9 | **Build a small LLM from scratch** | 10 |
| 10–11 | **MLOps & Deployment** (vLLM, CI/CD, monitoring) | 11 |
| 12 | **Advanced system design + evals at scale** + flagship capstone | 12 |

**Outcome:** Competitive for **LLM Engineer** and **mid-level AI Engineer** roles. 20+ projects, a from-scratch LLM, a fine-tuned model, and a production service.

---

## 🟠 18 Months — Senior-Track AI Engineer

**Goal:** Depth + breadth + production scale + a specialization track.

- Months 1–12: the 12-month plan, but with **deeper projects** and contributions to OSS.
- Months 13–15: **Specialize** — pick one: *LLM/Model* (advanced fine-tuning, RLHF/GRPO, distillation, custom architectures) **or** *Platform/MLOps* (K8s, distributed serving, eval platforms, FinOps).
- Months 16–18: **Capstone platform** (Phase 12 "AI Platform"), open-source contributions, write technical blog posts, mock interviews, and a polished public portfolio + talks.

**Outcome:** Competitive for **senior AI Engineer / senior LLM Engineer**, with demonstrable depth, scale experience, and thought leadership.

---

## 🔴 24 Months — Expert / Staff-Track

**Goal:** Mastery, leadership, and a recognized public footprint.

- Months 1–18: the 18-month plan with consistent shipping.
- Months 19–24:
  - Lead a substantial real-world AI system (work or large OSS).
  - Contribute meaningfully to a major OSS project (vLLM, LangChain, HF, etc.).
  - Publish: blog series, conference talk (AI Engineer Summit-style), or a paper/reproduction.
  - Mentor others; do system-design and on-call for AI services.
  - Stay on the frontier (reasoning RL, multimodal, efficient architectures).

**Outcome:** Staff/Principal-track AI Engineer or specialized expert (RAG/Agents/LLMOps/Model training).

## ⚠️ Timeline Realism Notes
- These assume **consistent** 20 hrs/week and **building** (not just watching). Passive learners take 2–3× longer.
- **Apply before you feel "ready."** Interview feedback is part of the curriculum; start applying ~1 month before your target.
- Depth in fundamentals (math/DL/from-scratch) is what separates the 6-month "app builder" from the 12-month "engineer" — don't skip it permanently, just sequence it after employability.
